
## 1 品位：如何全面高效地构建知识体系

1. 阅读“正统”文献或作品 -> 开始
    - 正统未必正确
    - 基准线思维
2. 阅读权威的综述文章
3. 读有趣的专著

## 2 如何回答好阅读理解
1. 宏观 -> 主题思想，作者态度 （换位思考）
2. 细节 -> 特定内容，具体信息 （7WH）

3. 主观 -> 看讲述人的信誉，并注意得出结论的逻辑
4. 客观 -> 考察信息源的可靠程度

注意：
英语写作中，不建议使用"I think"

## 3 速度：阅读瓶颈

1. 人脑接收信息的带宽 100-200 bits / 12-25汉字
2. 眼睛和大脑一同识别文字图像的速度

3. 调整书本位置
4. 语音辅助阅读（默读即可）-> 眼睛和大脑的同步
5. 不认识先跳过
